readme_content = """# VIPThink Checker (小老师空值检查员)

这是一个专为 VIPThink 课件管理系统量身定制的 Chrome 浏览器扩展插件。它的主要功能是自动化检查课件列表中的“小老师作品”配置，精准找出漏填“建议主题”或“录制建议”的课件，并自动生成校验报告。

## ✨ 功能特点

* **解放双手，全自动排查**：一键自动遍历当前页面的所有课件，模拟人工点击“更多” -> “小老师作品”，并自动开闭弹窗，全程无需人工干预。
* **精准突破框架限制**：
  * **幽灵菜单定位**：完美适配 Element UI 的底层逻辑，精准锁定挂载在页面底层的动态下拉菜单（处理 `offsetParent !== null` 只寻找肉眼可见的菜单）。
  * **富文本空值检测**：针对 Quill 编辑器进行深度过滤，自动剔除零宽空格（`\\u200B`）等不可见字符，绝不误判“假内容”。
* **自动化报告导出**：排查结束后，如果发现有漏填项，会自动生成并下载一份 `.csv` 格式的稽核报告；如果全部合格，则会弹出成功提示。
