# 课程封面图片处理工具

这个文件夹里的工具用于处理课程 CSV 数据：

1. 从 CSV 的图片 URL 下载课程封面
2. 把 CSV 单元格里的真实换行转换成 `<p>...</p>` 段落格式
3. 按课件代码匹配本地图片，并生成插入图片后的 Excel 文件

## 使用方法

1. 把需要处理的 CSV 文件放到当前文件夹。
2. 双击运行 `一键完整处理.bat`。
3. 每一步完成后，窗口会提示按任意键继续。
4. 最终会生成一个 `*_已插入课程封面.xlsx` 文件。

## CSV 列要求

CSV 中需要包含以下列：

- `列表_课件代码`：用于匹配图片文件名
- `列表_课程封面`：图片 URL，用于下载封面图片

如果 CSV 中没有 `课程封面图片` 列，插入图片程序会自动新增这一列。

## 输出文件

- 下载的图片会保存到 `下载的课程封面` 文件夹。
- 换行转换后的 CSV 会保存为 `原文件名_换行已转p.csv`。
- 插入图片后的 Excel 会保存为 `原文件名_已插入课程封面.xlsx`。

## 单独运行某一步

如果只想运行某个步骤，也可以双击对应文件：

- `运行下载.bat`：只下载图片
- `检查并转换换行为p标签.bat`：只转换换行
- `插入图片到表格.bat`：只生成插入图片后的 Excel

## 注意事项

- 程序会自动读取当前文件夹中最新的 CSV 文件。
- 图片文件名需要和课件代码一致，例如 `s4_v8_01_YY.jpg`。
- CSV 可以是逗号分隔或 Tab 分隔，程序会自动识别。
- 原始 CSV 不会被覆盖。
