# 豌豆課件小助手 (VIPThink Scraper Extension)

這是一個輕量級的 Chrome 瀏覽器擴充功能（插件），專為提取 VIPThink 課件列表與詳情數據而設計。它可以直接在瀏覽器原生環境中運行，無需安裝 Python 或其他爬蟲環境，且能完美繞過前端框架的 DOM 渲染延遲與圖層遮擋問題。

## ✨ 功能特點

* **一鍵無痕抓取**：點擊擴充功能按鈕，自動執行頁面滾動、跳轉與數據提取。
* **精準地毯式搜索**：自動遍歷並尋找 Vue 框架下動態生成的隱藏輸入框，確保數據 100% 準確提取。
* **原生 DOM 操作**：使用原生 JavaScript 觸發點擊，完美解決 Element UI 等框架的「圖層攔截」問題。
* **開箱即用**：無需配置任何環境，載入 Chrome 即可運行。
